### 温馨提示：

Lean版的默认用户名是root, 密码是password

Lienol版默认用户名是root, 密码为空
烧制完固件插入tf卡并启动完成，电脑端显示“网络（已连接）”之后，在浏览器输入 http://friendlywrt/ 可以直接打开路由器后台，无需修改本地连接设置或者查看IP地址。
